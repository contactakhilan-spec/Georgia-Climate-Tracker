## Tech Stack & Methodology
* **Language:** Python 3.x
* **Machine Learning:** `scikit-learn` (Trained a supervised **Linear Regression model** to calculate multi-decade climate warming coefficients)
* **Data Engineering:** `pandas` (Utilized optimized data manipulation techniques to parse and clean historical climate datasets)
* **Interactive Frontend:** Deployed using `Plotly Express` line chart layering to deliver an interactive, fast-loading browser dashboard.
